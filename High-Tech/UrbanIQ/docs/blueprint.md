# **App Name**: UranIQ

## Core Features:

- User Registration & Authentication: Secure signup and login using username, Aadhaar number, and password.
- Issue Reporting System: Citizens can submit problems categorized by sector with image upload, location tagging, and descriptions.
- Location-Based Services: GPS integration for precise problem location mapping using Google Maps API or Mapbox.
- Government Response Portal: Officials can respond to citizen reports with project details, timelines, budgets, and contractor information.
- Sector-Based Categorization: Seven predefined sectors for organized issue classification (Food, Road, Water, Women, Health, Municipal, Police).
- Government Accomplishments Dashboard: Showcasing completed projects and government achievements.
- Content Generation: Suggest improvements to citizens' descriptions or officials' responses via a tool powered by AI

## Style Guidelines:

- Primary color: Deep Blue (#1E40AF) representing trust and government.
- Background color: Light Blue (#D3DDEB) as a desaturated tone of blue to give a clean backdrop.
- Accent color: Yellow (#F7B32B) for call-to-action buttons and highlights; to guide users towards actionable items.
- Headings font: 'Inter' sans-serif for a clean, modern look.
- Body text: System fonts (optimized for readability across devices).
- Use clear, recognizable icons for each sector and common actions.
- Modern card-based layout with a clear visual hierarchy for easy information consumption.
